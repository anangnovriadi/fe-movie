# Movie Search

This is a Movie Search Project

* Demo Web: [fe-movie-rose.vercel.app](https://fe-movie-rose.vercel.app)

## Getting Started

### 1. Install & run the development server:

```bash
npm install
```

```bash
cp .env .env.example
```

```bash
npm run dev
```

### 2. Running Test

```bash
npm run test
```

### 3. Build Project

```bash
npm run build
```
